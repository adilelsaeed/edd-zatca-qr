== Description ==

Generate Zatca compatible invoices by including Zatca receipt QR to Easy Digital Downloads Orders

This plugin will add Zatca compatible QR to:
* order details (Receipt) page
* order email receipt
* With options to toggle that on/off

Add your company name and tax number to the plugin setting (found on EDD extensions setting) and enable the setting to start showing QR.
