## Description

**EDD Zatca QR Fatora** allows you to generate Zatca-compliant invoices with a QR code embedded directly into your Easy Digital Downloads (EDD) orders. This plugin ensures your invoices meet Saudi Arabia's Zakat, Tax, and Customs Authority (ZATCA) e-invoicing requirements. 

### Features:
- Automatically adds a ZATCA-compatible QR code to:
  - **Order Details (Receipt) Page**
  - **Order Email Receipts**
- Toggle QR code visibility for receipts and emails via plugin settings.
- Customizable plugin settings:
  - Input your **Company Name** and **Tax Number** to personalize the QR code.

## Installation

1. **Download the Plugin:**  
   Download the plugin files from the [GitHub repository](https://github.com/adilelsaeed/edd-zatca-qr) or [plugin page](https://adilelsaeed.com/ar/edd-zatca/).

2. **Upload the Plugin:**  
   - Navigate to `Plugins > Add New > Upload Plugin` in your WordPress admin area.
   - Select the downloaded ZIP file and click **Install Now**.

3. **Activate the Plugin:**  
   - Once installed, click the **Activate** button.

4. **Configure Settings:**  
   - Go to `EDD > Extensions > Zatca Settings`.
   - Add your **Company Name** and **Tax Number**.
   - Enable or disable the QR code for order receipts and emails.
